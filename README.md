  >事件捕获(event capturing)是从根节点开始从外向内的事件传播，而事件冒泡(dubbed  bubbling)则是从内到外的事件传播。<br>
  >可以参考该文章[DOM事件机制解惑](http://coderlt.coding.me/2016/11/22/js-event/)的中的图，可以看出DOM事件流的捕获和冒泡的过程
